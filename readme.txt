== Description ==
This wil not remove posts functionality but remove the link from the Admin menu. 
Use it when using custom post types and not the standard posts

You can find the plugin home page (and leave comments) [here](http://reciprocity.be/cms-menu/).

== Installation ==
1. Upload `remove-posts-from-admin` to the `/wp-content/plugins/` directory. 
2. Activate `CMS-Like Admin Menu` through the 'Plugins' menu in WordPress.
